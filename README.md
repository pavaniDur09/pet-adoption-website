# Pet Adoption & Charity Website

A responsive multi-page website designed to support a pet adoption and charity initiative. The website provides information about the organisation, available pets for adoption, services offered, and allows users to submit enquiries and donation details through interactive forms.

## Project Overview
This website was developed to promote pet adoption and raise awareness for animal welfare. It provides a user-friendly interface where visitors can browse available pets, learn about services, and contact the organisation.

The project focuses on accessibility, responsive design, and intuitive navigation to ensure a positive user experience.

## Features
- Multi-page website with structured navigation  
- Pet adoption listings with clear presentation  
- Contact form for enquiries  
- Donation form for supporter contributions  
- Responsive layout for mobile, tablet, and desktop devices  
- Accessible and user-friendly design  

## Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- Responsive design techniques
